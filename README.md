# Coupon Management - Option A (Full eligibility)
## Overview
Node.js + Express in-memory coupon service implementing the Assignment B requirements:
- POST /coupons (create)
- GET /coupons (list)
- POST /coupons/best (best coupon)
- POST /coupons/apply (apply and increment usage)

## Run
1. npm install
2. npm start
Server: http://localhost:3000

## Example
Create coupon: POST /coupons
Find best: POST /coupons/best
Apply: POST /coupons/apply
